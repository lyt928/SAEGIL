# AI 기반 공사현장 사고 예방 및 실시간 경보 시스템

공사현장 영상을 실시간으로 분석하여 작업자, 보호장비, 위험구역, 주요 위험상황을 감지하고 즉시 경고를 발생시키는 AI 기반 안전관리 프로토타입입니다.

이 프로젝트의 핵심 목표는 사고 발생 후 대응이 아니라, 사고 전 위험 징후를 조기에 탐지하여 안전사고를 예방하는 것입니다.

현재 구현 및 1차 개발 범위는 CCTV형 감시 시나리오를 기준으로 하며, 장비부착형 시나리오는 향후 확장안으로 다룹니다.

---

## 1. 프로젝트 개요

건설 현장은 추락, 협착, 낙하물, 장비 충돌 등 다양한 위험요소가 상시 존재하는 환경입니다. 기존의 안전관리 방식은 관리자의 순회 점검이나 CCTV 수동 확인에 의존하는 경우가 많아, 실시간 감시와 사고 전조 탐지에 한계가 있습니다.

본 프로젝트는 AI 기반 영상 분석을 통해 이러한 한계를 보완하고, 현장에서 즉시 활용 가능한 안전관리 시스템을 구현하는 것을 목표로 합니다.

### 목표

- 작업자 실시간 탐지
- 안전모·안전조끼 등 보호장비 착용 여부 판별
- 위험구역 침입 감지
- 위험상황 조기 판단
- 화면 경고 및 알림음 기반 실시간 경보
- 위험 이벤트 로그 기록 및 모니터링 화면 제공

---

## 2. 핵심 기능

### 2.1 작업자 및 보호장비 인식

카메라 영상에서 작업자를 탐지하고, 안전모·안전조끼 등 주요 보호장비 착용 여부를 판별합니다. 미착용 작업자가 발견되면 즉시 경고를 발생시킵니다.

### 2.2 위험구역 침입 감지

영상 내 특정 영역을 가상 위험구역으로 설정하고, 작업자가 해당 구역에 접근하거나 진입할 경우 실시간으로 감지합니다. 예를 들어 굴착기 작업 반경, 개구부 주변, 고소작업 구역 등을 위험구역으로 지정할 수 있습니다.

### 2.3 위험상황 판단

단순 객체 탐지를 넘어, 객체 간 위치 관계와 작업자의 상태를 바탕으로 추락, 협착, 낙하물 위험 등 사고 가능성이 높은 상황을 조기에 감지합니다.

초기 구현은 규칙 기반으로 설계하고, 필요 시 자세 추정이나 보조 분석 모델을 확장할 수 있습니다.

### 2.4 실시간 모니터링 및 경보

실시간 영상, 탐지 결과, 경고 상태, 위험 이벤트 로그를 한 화면에서 확인할 수 있는 인터페이스를 제공합니다. 경고는 화면 문구, 상태 표시, 알림음 또는 부저 형태로 출력됩니다.

## 6. 모듈 설명

### `camera`

- USB/IP 카메라 스트림 수집
- 해상도 및 FPS 설정
- 프레임 전달

### `detection`

- 작업자, 안전모, 안전조끼 탐지
- YOLO 추론 결과 표준화

### `ppe`

- 작업자와 보호장비 bbox 매칭
- 착용 여부 판정
- 미착용 이벤트 생성

### `zones`

- polygon 또는 rectangle 기반 위험구역 정의
- 중심점/교차 판정으로 진입 여부 계산

### `risk`

- 객체 간 거리, 위치 관계, 구역 침입 여부로 위험 판단
- 초기 버전은 규칙 기반

### `alerts`

- 화면 오버레이
- 상태 패널 갱신
- 알림음/부저 제어

### `logging`

- 이벤트 로그 저장
- SQLite 또는 JSONL 기반 기록

### `dashboard`

- 실시간 영상
- bbox 및 라벨 표시
- 현재 경고 상태
- 최근 이벤트 목록 표시
