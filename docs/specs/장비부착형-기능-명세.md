# 장비 부착형 기능 명세

## 1. 목적

포크레인, 지게차, 굴착기 등 대형 장비에 카메라를 부착해 장비 주변의 작업자를 감지하고, 장비와 사람 사이의 근접 충돌 위험을 실시간으로 경고합니다.

이 방식은 구역 감시보다 장비 중심 사고 예방에 적합합니다.

## 2. 주요 사용 시나리오

1. 장비 전방에 작업자가 접근하는 경우 경고
2. 후진 중 장비 후방 사각지대의 작업자를 감지
3. 회전 중인 장비의 작업 반경 내 인원을 감지
4. 붐, 암, 버킷 작업 범위에 사람이 들어오는 경우 경고
5. 운전자 화면 또는 경고음으로 즉시 위험을 전달

## 3. 필수 기능

### 3.1 주변 작업자 감지

- 장비 기준 전방, 후방, 측면의 사람 감지
- 근거리 우선 감지 튜닝
- 프레임 간 추적 ID 유지

### 3.2 거리 기반 위험 판단

- 장비와 작업자 간 상대 거리 추정
- 거리 단계별 경고 레벨 적용
- 너무 가까운 경우 즉시 고위험 경고

### 3.3 방향 기반 위험 판단

- 전진 방향 위험영역 계산
- 후진 방향 위험영역 계산
- 회전 방향 및 회전 반경 위험영역 계산

### 3.4 작업 반경 침범 감지

- 붐/암/버킷 예상 작업영역 침범 감지
- 장비 종류별 위험영역 템플릿 구성
- 장비 상태 정보 연동 시 정확도 향상

### 3.5 운전자 경고 인터페이스

- 화면 경고
- 단계별 경고음
- 방향성 표시
- 고위험 시 즉시 주의 문구

### 3.6 카메라 상태 감시

- 카메라 가림 감지
- 렌즈 오염 감지
- 야간 또는 역광으로 인한 품질 저하 감시

### 3.7 이벤트 기록

- 장비 ID
- 카메라 위치
- 접근 방향
- 위험 거리
- 이벤트 시각
- 스냅샷 또는 영상 클립

## 4. 권장 경고 규칙

- 전방 5m 이내 인원: 주의
- 전방 3m 이내 인원: 경고
- 후방 사각지대 인원: 즉시 경고
- 회전 반경 침범: 높음 경고
- 작업 반경 + 근접 거리 동시 만족: 매우 높음 경고

## 5. 운전자 화면에 필요한 항목

- 현재 카메라 영상
- 위험 방향 표시
- 근접 인원 수
- 가장 가까운 인원 거리
- 경고 등급
- 카메라 이상 상태 표시

## 6. 로그 권장 항목

- `event_type`
- `severity`
- `equipment_id`
- `camera_position`
- `track_id`
- `distance_estimate`
- `relative_direction`
- `bbox`
- `snapshot_path`
- `clip_path`
- `created_at`

## 7. 개발 우선순위

1. 사람 감지 안정화
2. 전방/후방 근접 경고
3. 후진 사각지대 경고
4. 방향 기반 위험영역 계산
5. 회전 반경 침범 감지
6. 운전자용 UI
7. 장비 상태 연동

## 8. 도입 시 장점과 한계

장점:
- 장비와 작업자 충돌 사고 예방 효과가 직접적임
- CCTV 사각지대를 보완할 수 있음
- 차별화 포인트가 분명함

한계:
- 화면 흔들림과 진동 대응이 필요함
- 먼지, 오염, 역광 환경 영향을 크게 받음
- 동적 위험영역 계산이 필요해 구현 난도가 높음
