# CCTV형 기능 명세

## 1. 목적

고정형 CCTV 또는 고정형 네트워크 카메라를 이용해 현장 구역을 상시 감시하고, 작업자 안전장비 착용 상태와 위험구역 접근 여부를 실시간으로 감지합니다.

이 방식은 현장 전체를 넓게 보는 데 유리하며, 현재 프로젝트 구조와도 가장 잘 맞습니다.

## 2. 주요 사용 시나리오

1. 출입금지 구역에 작업자가 들어오는지 감지
2. 안전모, 안전조끼 미착용자를 감지
3. 위험구역에 일정 시간 이상 머무르는 작업자를 감지
4. 야간이나 저조도 환경에서도 경고 이벤트를 기록
5. 관리자 화면에서 이벤트 이력과 스냅샷을 확인

## 3. 필수 기능

### 3.1 작업자 감지

- 사람 객체 탐지
- 다중 작업자 동시 감지
- 추후 트래킹 ID 기반 연속 추적 확장

### 3.2 PPE 감지

- 안전모 착용 여부 판단
- 안전조끼 착용 여부 판단
- 추후 안전화, 하네스 확장 가능 구조 유지

### 3.3 위험구역 감지

- polygon 기반 위험구역 설정
- 작업자 발 위치 기준 진입 판정
- 구역별 위험도 설정

### 3.4 체류 시간 감지

- 위험구역 진입 시각 기록
- `N초 이상 체류` 시 경고 상향
- 구역별 체류 허용 시간 설정 가능

### 3.5 경고 출력

- 화면 오버레이 경고 문구
- 경광등 또는 부저 연동 가능 구조
- 관리자 대시보드 알림 표시

### 3.6 이벤트 기록

- 이벤트 코드
- 발생 시각
- 카메라 ID
- 작업자 추적 ID
- 위험구역 ID
- 스냅샷 또는 클립 경로

## 4. 권장 경고 규칙

- 안전모 미착용: 중간 경고
- 안전조끼 미착용: 낮음 경고
- 위험구역 진입: 높음 경고
- 위험구역 장기 체류: 매우 높음 경고
- PPE 미착용 + 위험구역 진입: 복합 경고

## 5. 관리자 화면에 필요한 항목

- 실시간 영상
- 현재 경고 목록
- 최근 이벤트 목록
- 이벤트 상세 정보
- 스냅샷 또는 영상 클립 재생
- 카메라별 상태 표시

## 6. 로그 권장 항목

- `event_type`
- `severity`
- `camera_id`
- `track_id`
- `zone_id`
- `bbox`
- `confidence`
- `snapshot_path`
- `clip_path`
- `created_at`

## 7. 개발 우선순위

1. 사람 감지
2. 안전모, 안전조끼 판정
3. 위험구역 진입 감지
4. 체류 시간 감지
5. 이벤트 로그 저장
6. 대시보드 조회
7. 경광등/부저 연동

## 8. 도입 시 장점과 한계

장점:
- 넓은 구역 감시가 가능함
- 현재 코드 구조와 잘 맞음
- 초기 실증이 쉬움

한계:
- 사각지대가 생길 수 있음
- 사람이나 자재에 가려질 수 있음
- 장비 바로 옆의 근접 충돌은 놓칠 수 있음
